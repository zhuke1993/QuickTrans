# 🚀 快速开始指南

## 三步完成安装

### 第一步：加载插件到Chrome

1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择本项目根目录
6. 看到 **"AI翻译助手"** 出现在扩展列表中

### 第二步：配置API（必需）

1. 点击扩展列表中的 **"详细信息"**
2. 点击 **"扩展程序选项"**
3. 在设置页面点击 **"添加新配置"**
4. 填写表单：
   ```
   配置名称：OpenAI GPT-3.5
   API端点：https://api.openai.com/v1/chat/completions
   API密钥：sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
5. 点击 **"测试连接"** 验证
6. 点击 **"保存配置"**

### 第三步：开始使用

1. 打开任意网页（如：https://en.wikipedia.org）
2. 用鼠标选中一段英文
3. 等待0.5秒，自动显示翻译弹窗
4. 或点击紫色翻译图标
5. 享受翻译！🎉

## 📝 获取OpenAI API Key

### 如果你已有OpenAI账号：

1. 访问：https://platform.openai.com/api-keys
2. 点击 **"Create new secret key"**
3. 复制生成的密钥（以sk-开头）
4. 在插件设置中粘贴密钥

### 如果你没有OpenAI账号：

1. 访问：https://platform.openai.com/signup
2. 注册账号并验证
3. 添加支付方式
4. 创建API密钥（步骤同上）

### 替代方案 - 使用其他API服务：

本插件支持所有兼容OpenAI协议的API服务，包括：

- 国内API代理服务（更快速、更便宜）
- Azure OpenAI
- 自建OpenAI兼容服务

只需将API端点和密钥换成对应服务的即可。

## ⚙️ 可选：调整偏好设置

在设置页面的 **"偏好设置"** 部分：

### 默认目标语言
- 设置你常用的翻译目标语言
- 默认为：中文

### 自动显示翻译弹窗
- ✅ 开启：选中文本后自动显示翻译（推荐）
- ❌ 关闭：需要点击图标才显示翻译

## 🎯 使用技巧

### 技巧1：切换目标语言
在翻译弹窗中，点击目标语言下拉框，选择新语言，自动重新翻译。

### 技巧2：复制翻译结果
点击弹窗底部的 **"复制译文"** 按钮，翻译结果立即复制到剪贴板。

### 技巧3：快速关闭弹窗
- 按 `ESC` 键
- 点击弹窗外部任意区域
- 点击右上角 `×` 按钮

### 技巧4：管理多个API配置
1. 在设置页面添加多个不同的API配置
2. 根据需要切换激活的配置
3. 例如：工作用OpenAI，个人用国内代理

## 🐛 遇到问题？

### 问题1：翻译失败，提示"未配置API"
**解决**：请先按照第二步配置API。

### 问题2：提示"API密钥无效"
**检查**：
- API密钥是否正确（注意不要有多余空格）
- OpenAI账户是否有余额
- 网络连接是否正常

### 问题3：翻译很慢
**原因**：取决于API服务的响应速度和网络质量。
**建议**：使用更快的API服务，或检查网络连接。

### 问题4：某些网站无法使用
**说明**：部分特殊网站可能有兼容性问题。
**反馈**：请在GitHub上提交Issue。
